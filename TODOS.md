# 缺陷和待办事项

## 已知缺陷（Known Issues）

### 缺陷 #1: 移动设备上侧边对比不够友好
- **严重程度**: 中
- **影响范围**: 手机/平板用户体验
- **现象**: 屏幕宽度小于 768px 时，左右对比代码块重叠
- **临时解决方案**: 使用桌面浏览器或旋转设备到横屏模式
- **计划修复**: v1.1.0 - 添加响应式单栏模式

### 缺陷 #2: 大文件（>500 行）加载性能慢
- **严重程度**: 中
- **影响范围**: 对比大型文件时
- **现象**: 首屏加载延迟 2-3 秒，滚动不流畅
- **根因**: 一次性 DOM 渲染所有代码行
- **临时解决方案**: 分割代码成多个小文件
- **计划修复**: v1.1.0 - 实现虚拟滚动
- **预计工作量**: 8-10 小时

### 缺陷 #3: 语法高亮对某些语言不完整
- **严重程度**: 低
- **影响范围**: 非主流编程语言
- **现象**: Go、Rust 等语言的关键字高亮不全
- **原因**: 正则表达式规则不完整
- **临时解决方案**: 手动修改 HTML 中的高亮规则
- **计划修复**: v1.2.0 - 扩展语言支持

### 缺陷 #4: 敏感信息过滤不够全面
- **严重程度**: 高
- **影响范围**: 包含密钥/密码的代码
- **现象**: API Key、密码可能在对比中泄露
- **原因**: 过滤规则基础
- **临时解决方案**: 手动审查和编辑生成的 HTML
- **计划修复**: v1.1.0 - 增强敏感信息检测
- **预计工作量**: 4-6 小时
